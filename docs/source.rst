SOURCE
======

BOTLIB provides the following modules:

.. autosummary::
    :toctree: 
    :template: module.rst

    bus          - messaging
    clk          - clock/repeater
    cms          - commands
    csl          - console
    dbs          - databases
    hdl          - handler
    irc          - internet relay chat
    obj          - objects
    ofn          - object functions
    prs          - parser
    rss          - rich site syndicate
    thr          - threads
    trm          - terminal
    udp          - udp to irc relay
