SOURCE
======

BOTLIB contains the following modules:

.. autosummary::
    :toctree: 
    :template: module.rst

    bot.clk             - clock/repeater
    bot.cmd             - commands
    bot.csl             - console
    bot.dbs             - database
    bot.err             - errors
    bot.flt             - list of bots
    bot.hdl             - handler
    bot.irc             - internet relay chat
    bot.isp             - introspect
    bot.krn             - core handler
    bot.obj             - base classes
    bot.opr             - opers
    bot.mbx             - email
    bot.prs             - parse
    bot.spc             - specifications
    bot.tsk             - tasks
    bot.tms             - time
    bot.trc             - trace
    bot.udp             - udp to channel
    bot.usr             - users
    bot.utl             - utilities

BOTD itself provides 1 module:

.. autosummary::
    :toctree: 
    :template: module.rst


    botd.rss             - rich site syndicate

You can add you own modules to the botd package, its a namespace package.
